# AKSIBGUU

Мобильное приложение для Аэрокосмического колледжа СибГУ им. академика М. Ф. Решетнёва , разработанное на **Flutter** и языке Dart.  
Проект используется как часть учебной/дипломной работы по направлению «Информационные системы и программирование».

## Описание

Приложение предназначено для абитуриентов и студентов Аэрокосмического колледжа. Оно предоставляет информацию о колледже, специальностях, расписании, вакансиях, а также позволяет подать документы на поступление.

## Основные возможности

### Для гостей (абитуриентов)
- **Анимированный Splash-экран** — красивая анимация при входе в приложение с логотипом колледжа
- **Абитуриенту** — подробная информация о колледже, 15 специальностей с описанием, квалификацией, карьерными перспективами и зарплатами
- **Профориентационный тест** — 15 вопросов для подбора подходящей специальности
- **Подача документов** — форма для онлайн-подачи документов на поступление с загрузкой файлов (PDF, JPG, PNG)
- **Дополнительное образование** — курсы и программы повышения квалификации
- **Контакты** — контактная информация колледжа, телефоны, email, информация о директоре
- **Вход в личный кабинет** — авторизация для перехода в режим студента

### Для студентов
- **Главная** — лента новостей и историй колледжа с полноэкранным просмотром (аналог Stories)
- **Расписание** — расписание звонков и расписание занятий по группам
- **Вакансии** — актуальные вакансии с возможностью отклика
- **Контакты** — контактная информация колледжа
- **Профиль** — личный кабинет студента, портфолио, создание резюме

Профиль — вход в личный кабинет

## Запуск проекта

1. Установить Flutter SDK:  
   https://docs.flutter.dev/get-started/install

   
3. Клонировать репозиторий:
```bash
  git clone https://github.com/ZZZzzz-228/aksibguu.git
   cd aksibguu

   
3. Установить зависимости:
    flutter pub get

    
4. Запустить приложение:
    flutter run

    
5. Сборка APK
    flutter build apk --split-per-abi
```
